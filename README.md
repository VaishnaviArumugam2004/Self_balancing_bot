A self-balancing robot built using embedded systems and control algorithms to maintain stability on two wheels, similar to an inverted pendulum system.
This project demonstrates real-time balance control using sensor feedback and motor actuation. The robot continuously adjusts its position to stay upright.

Features
* Real-time angle detection
* PID-based stabilization
* Continuous motor control
* Compact embedded implementation

Working Principle
The robot uses sensor data (such as an accelerometer and gyroscope) to detect its tilt angle. A control algorithm (PID) calculates the required correction, and motors are driven accordingly to maintain balance.
